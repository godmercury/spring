1. C
    컴파일러 언어로 작고 빠른 프로그램
    어셈블리어와 고급 언어의 장접을 가지고 있다
    뛰어난 이식성
    주 사용처
     - 시스템 프로그램(운영체제, 디버깅 S/W, 미들웨어) 비트 조작, 메모리 접근, 형 변환
     - 컴파일러, 유틸리티
     - 응용 프로그램(게임, 워드프로세서, 오라클)
     - 임베디드 프로그래밍


2. C++
    객체지향 언어로 C의 확장판
    다중 상속, 오버로드
    주 사용처
     - 게임 프로그램
     - 빠른 처리가 필요한 응용프로그램
     - 하드웨어 제어 프로그램
    

3. C#
    객체지항언어 (C언어의 기본문법 , C++의 객체지항개념, UI 컴포넌트)
    .NET 프레임워크에서 개발
    이기종 간에 인터넷을 통한 자료 교환을 위해 개발
    주 사용처
     - 웹 응용프로그램
     - 네트워크 프로그램


4. JAVA
    C++과 유사하나 모호하거나 복잡한 기능을 제거한 객체지향 프로그램
    모든 데이터는 객체를 통하여 처리
    바이트 코드로 컴파일되기 때문에 자바를 지원하는 어떠한 플랫폼에서도 실행
    주 사용처
     - 웹 응용 프로그램
     - 분산환경에 적합한 언어
     - 휴대기기용 프로그램


5. Python
    인간 친화적 언어로 익히기 쉬운 언어
    다른 언어와 혼합하여 개발 가능
    인터프리터식 언어
    주 사용처
     - C/C++과의 결합
     - 웹 프로그래밍
     - GUI 프로그래밍


6. SWIFT
    객체지향 언어
    주 사용처
     - iOS 애플리케이션


7. Javascript
    객체기반의 스크립트 언어
    인터프리터식 언어
    주 사용처
    - 웹 프로그래밍
    - 모바일 애플리케이션

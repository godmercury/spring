개발툴

1. Visual Studio Code
    장점
      - 무료
      - 가볍다
      - 다양한 플로그인 및 확장 프로그램
      - 자동완성, 포맷팅, 디버깅 툴 지원
      - 다양한 언어 지원
    
    단점
    - 자동 완성등의 버그 발생 시 앱을 재시작해야 함
    - 자체적으로 지원하는 기능은 매우 부족하여 필요한 플러그인을 설치해야 한다

2. Eclipse
    장점
    - 무료
    - 여러 프로젝트를 동시에 관리할 수 있다.

    단점
    - 무겁다.
    - 지속적인 패치에도 자잘한 버그들이 많다.
    - JAVA를 제외한 다른 언어는 개발하기 불편하다.

3. IntelliJ
    장점
    - 우수한 스마트 코드 작성 기능들 덕분에 가발자의 생산성을 대폭 늘려준다.
    - 자동완성, 소스코드분석, 지능형리펙토링, 디버거, 테스트러너 기능이 이클립스와 비교하여 우수하다.
    - 형상관리를 git으로 한다면 intelliJ가 편함

    단점
    - 유료
    - 무겁다
    - eclipse로 주로 이루어지는 프로젝트 개발환경과 맞지 않는 경우가 존재함.
